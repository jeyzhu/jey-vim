jey-vim
=======
##安装方法

    1.windows 请绕行
    2.需要已安装VIM
    3.需要已安装git
    4.git clone https://github.com/jeyzhu/jey-vim.git
    5.执行 ./install.sh
    6.根据脚本提示输入参数
    7.安装当中会自行打开VIM安装插件 请耐心等待VIM自行关闭
    
##特色说明
    1.插件配置与VIMRC配置分离
    2.可自定义VIMRC用户自己的信息 无需修改VIMRC
    3.SWAP与无限撤销文件操作 保存路径自定义 避免有时将.swap文件提交
    4.版本更新只需更新GIT就能体验新的功能 无需重新配置VIMRC
    5.VIM插件自动安装
